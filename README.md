## Step: 1
Clone the repository
```sh 
git clone git@github.com:mahmudulhsn/crud-laravel-app.git
```
## Step: 2
Go to the project directory
```sh
cd crud-laravel-app
```
## Step: 3
Run the command to create the .env file
```sh 
    cp .env.example .env
```
## Step: 4
Run the command
```sh 
    make setup
```

## Step: 5
Then run the command
```sh 
    make key
```
## Step: 6
Then run the command
```sh 
    make migrate
```
## Step: 7
Then run the command
```sh 
    make seed
```
Now your project is ready to run.
